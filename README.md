# Beauty Helper — Frontend

Frontend for a SaaS CRM platform for beauty professionals. Provides a modern dashboard interface for managing clients, appointments, and business operations.

## Stack

Next.js, TypeScript, React, TanStack Query, Tailwind CSS, REST API

## Features

- Secure authentication (JWT, Google OAuth)
- Protected routes and session handling
- Dashboard with real-time data
- REST API integration
- Reusable component-based architecture
- Optimized data fetching and caching

## Architecture

Built with Next.js App Router and modern frontend patterns, focusing on scalability, performance, and maintainability.
